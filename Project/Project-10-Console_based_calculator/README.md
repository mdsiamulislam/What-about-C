# Console-Based Calculator

Welcome to the Console-Based Calculator project! This simple C program allows you to perform basic arithmetic calculations in a console environment.

## Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Modulo (`%`) - (integer modulo)
